#### v0.14.10 ‚Äî RR‚ÜíSE seeding fix

**Root cause** (`advancer.ts`): The snake seeding reversed group order for even ranks (runners-up), assigning group 1's runner-up ‚Üí seed 8. `bracketSeedOrder(8)` = [1,8,4,5,2,7,3,6] places seed 1 vs seed 8 in the same first-round match ‚Äî causing group 1 winner and runner-up to immediately face each other.

**Fix**: Removed snake reversal ‚Äî all ranks now use ascending group order. Seeds 1‚Äì4 = G1‚ÄìG4 winners; seeds 5‚Äì8 = G1‚ÄìG4 runners-up. `bracketSeedOrder` then pairs: 1v8 (G1W/G4R), 4v5 (G4W/G1R), 2v7 (G2W/G3R), 3v6 (G3W/G2R).

**`seedToGroupLabel`** (`bracket/page.tsx`): Removed the same even-rank reversal so placeholder labels match the corrected seeding.

**Tests** (`advancer.test.ts`): Updated rank-2 expectation from descending to ascending group order.
