# Project Overview

## Goal

Build a modern web application for the competitive table tennis community that combines:

- player profiles and ratings
- tournament management
- event creation
- bracket generation
- match result submission and confirmation
- automatic rating updates

This MVP must support a **complete vertical workflow** suitable for interviews and demos.

## Target Users

- Table tennis players
- Tournament directors
- Platform administrators

## Core Demo Narrative

The system should support this workflow:

1. User signs up
2. User creates player profile
3. User views ratings and match history
4. Tournament director creates a tournament
5. Director creates an event such as **U1800 Singles**
6. Players are entered into the event
7. System generates a bracket
8. Player submits match result with game scores
9. Opponent confirms using confirmation code
10. Bracket advances automatically
11. Ratings update
12. Player profile shows rating delta
