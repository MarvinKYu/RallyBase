# Acceptance Criteria

The MVP is complete when the following work end-to-end.

## Player Features

Users can:

- create accounts
- create player profiles
- view ratings
- search players
- view match history with rating delta

## Tournament Features

Directors can:

- create tournaments
- create events
- add entrants
- generate brackets

## Match Features

Players can:

- submit match results
- enter game-by-game scores
- confirm opponent result using code

## Ratings

Confirmed matches:

- update player ratings
- create rating transaction entries
